# Portfolio — Manuel Nunziata

Sito portfolio personale costruito durante il corso Full-Stack di **Epicode 2026**.  
Realizzato interamente a mano — nessun framework CSS, solo HTML semantico, CSS custom e JavaScript vanilla.

## Struttura

```
├── index.html        # Homepage con bio, competenze, esperienze, progetti e contatti
├── progetti.html     # Pagina dedicata ai progetti
└── assets/
    ├── css/style.css # Stile custom
    ├── js/script.js  # Logica modale e form di contatto
    └── img/          # Immagini (foto profilo)
```

## Sezioni

- **Chi sono** — breve presentazione con foto profilo
- **Competenze** — tabella con barre di progresso autovalutate (HTML5, CSS3, Git, VS Code, Markdown)
- **Esperienze** — cronologia lavorativa e formativa
- **Progetti** — raccolta dei lavori realizzati durante il corso
- **Contatti** — form modale con validazione e box di conferma

## Tecnologie usate

- HTML5 semantico
- CSS3 (selettori, pseudo-classi, box model, variabili CSS)
- JavaScript vanilla (DOM manipulation, eventi, form validation)
- [Bootstrap Icons](https://icons.getbootstrap.com/) — iconografia
- [Material Symbols](https://fonts.google.com/icons) — icone navigazione
- Git & GitHub — versionamento e deploy

## Come visualizzarlo

Apri `index.html` direttamente nel browser, oppure usa l'estensione **Live Server** di VS Code.
